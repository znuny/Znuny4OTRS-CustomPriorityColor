CustomPriorityColor
=================
With this extension you will be able to change ticket priority colors via SysConfig.

**Prerequisites**

- Znuny4OTRS-Repo

- OTRS 4.0

**Installation**

Download the package and install it via admin interface -> package manager or use Znuny4OTRS-Repo.

**Download**
